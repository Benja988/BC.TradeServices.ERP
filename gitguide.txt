Creating Projects and Issues
In your repo page -> Projects -> New project -> choose Board (or "Projects (beta)" depending on UI).
Name: Sprint Board - Week 1 or Project Board
Create columns

Issues CLI sample
gh issue create --title "Create project board & populate" --body "Create Kanban board with columns Backlog, To Do, In Progress, Code Review, Testing, Done. Add Week 1 issues to Backlog/To Do." --label "week 1"
